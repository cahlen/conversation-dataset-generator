Define the exact topic, personas (with descriptions!), scenario, style, and even specific keywords to include.

Generating Data for Specific Goals

Training a "Sitcom Banter" Style LoRA

Style

Create a LoRA that makes an LLM generate witty, observational dialogue reminiscent of a classic sitcom.

python generate.py \
  --num-examples 1000 \
  --topic "the absurdity of everyday errands" \
  --persona1 "Alex" --persona1-desc "slightly neurotic, prone to overthinking" \
  --persona2 "Sam" --persona2-desc "laid-back, responds with dry wit" \
  --scenario "waiting in line at the post office" \
  --style "observational, witty, fast-paced banter, like Seinfeld" \
  --output-file sitcom_style_dataset.jsonl

Training a "Helpful Coding Mentor" Persona LoRA

Persona

Fine-tune a model to act as a patient, encouraging coding mentor.

python generate.py \
  --num-examples 500 \
  --topic "debugging a common Python error" \
  --persona1 "MentorBot" --persona1-desc "patient, knowledgeable Python tutor" \
  --persona2 "Learner" --persona2-desc "beginner programmer feeling stuck" \
  --scenario "working through a coding problem online" \
  --style "supportive, clear, step-by-step" \
  --output-file mentor_persona_dataset.jsonl

Training a Topic-Focused LoRA ("Quantum Computing")

Topic

Make the LLM more fluent when explaining complex topics conversationally.

python generate.py \
  --num-examples 750 \
  --topic "basic concepts of quantum computing" \
  --persona1 "QuantumGuru" --persona1-desc "expert simplifying with analogies" \
  --persona2 "CuriousChris" --persona2-desc "intelligent but new to quantum" \
  --scenario "casual conversation over coffee" \
  --style "simplified, analogy-driven, patient" \
  --output-file quantum_topic_dataset.jsonl

Generating Dialogue with Web Search

Web Context

Create dialogue involving specific individuals by providing web search terms for context.

python generate.py \
  --creative-brief "Conversation between MKBHD and Stanley Kubrick" \
  --num-examples 5 \
  --persona1-search-term "Marques Brownlee tech review style" \
  --persona2-search-term "Stanley Kubrick film style" \
  --output-file mkbhd_kubrick.jsonl

Technical Details

Requirements

  • Python 3.8+
  • Key Libraries: torch, transformers, datasets, huggingface_hub
  • Optional: duckduckgo-search for web context
  • GPU recommended for faster generation

Installation

# Clone repo
git clone https://github.com/cahlen/conversation-dataset-generator.git
cd conversation-dataset-generator

# Create virtual environment
python3 -m venv venv
source venv/bin/activate # Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
